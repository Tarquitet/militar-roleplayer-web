## 🛠️ Technical Specifications & Versioning

### ⚙️ System Architecture

- [**Database Connection**](config/conexion.php): Secure PDO link to MySQL.
- [**Tactical Dictionary**](config/textos.php): Centralized string and alert management.
- [**Economic Config**](config/precios.php): Global asset cost logic by range and class.

### 🕹️ Operational Modules

- [**Central Command**](views/staff_dashboard.php): Global resource and jurisdiction control for Staff.
- [**Store & Tech Tree**](views/lider_tienda.php): Integrated acquisition system with "Fog of War".
- [**Operational Hangar**](views/lider_inventario.php): Stock management and naval fleet deployment.
- [**Intelligence Radar**](index.php): Public faction status monitoring in real-time.

### 📡 Environment Details

- **Hosting:** InfinityFree (PHP/MySQL Hosting).
- **Stack:** PHP 8.2+ | MySQL (MariaDB) | Tailwind CSS.
- **Styling:** [Custom Military CSS Framework](assets/css/military.css).
